segreg.model.alt.fit_one_bkpt¶

fit_one_bkpt(indep, dep, num_end_to_skip=3, verbose=False, extra_verbose=False, optimize=True, check_near_middle=False)¶

Estimate one-bkpt segmented regression model.

Notes

This implementation does not compute anything when there are 2n+1 distinct data points, and num_end_to_skip is n-1. In such a case, there is only one possible configuration, where the bkpt is at the nth data point.
